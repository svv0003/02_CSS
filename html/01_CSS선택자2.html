기본 속성 선택자

      - 태그에 작성된 특정 속성을 이용해 선책하는 선택자

      작성법
      
        선택자[속성명="속성값"]
          {css 코드;}

p1

자식 선택자(>)

      지정된 요소 바로 한 단계(1 level/depth) 하위 요소 선택

      작성법
      
        선택자1 > 선택자2 {
          CSS 코드;
        }

        - 선택자1 : 부모 요소(감싸고 있는 상위 요소)
        - 선택자2 : 자식 요소(속해져 있는 하위 요소)
    부모
  • 자식1
  • 자식2
  • 자식3
  • 자식4

후손 선택자(작성법 : 띄어쓰기)

      지정된 요소의 모든 하위 요소 선택 (n level 이하 모두)

      작성법
      
        선택자1 선택자2 {
          CSS 코드;
        }

        선택자1 : 부모(조상) 요소
        선택자2 : 후손 요소

반응 선택자

      사용자의 움직임에 반응하여 스타일이 변경되도록 하는 선택자

      선택자:active -> 선택된 요소가 클릭되고 있을 때
      선택자:hover -> 선택된 요소 위에 마우스가 올라 갔을 때
:active
:hover

상태 선택자

      입력 양식(input, input 관련 태그)의 상태에 따라 선택되는 선택자

      :focus  -> 요소에 초점이 맞춰졌을 때(input 요소를 클릭해서 입력 준비가 됐을 때)
      :checked  -> 요소가 체크 됐을 때(radio, chechbox)
      :enabled / disabled -> 요소가 활성화 / 비활성화 됐을 때
:focus
아이디 입력 :
:checked
:enabled / disabled

동의 선택자

      동위(동등한 위치 == 형제 관계) 관계에서 다음(뒤)에 위치한 요소를 선택하는 선택자
