CSS 선택자의 우선 순위

      기본적으로 css 속성은 style 태그 또는 css 파일에 작성된 순서(위 -> 아래)대로 해석이 진행되지만
      같은 요소에 여러 css 속성이 설정되는 경우 우선순위가 적용됨

      * 알아두어야 할 것 *
      1) 동일한 우선 순위로 css 속성이 설정된 경우
         -> 제일 마지막에 작성된 css 속성이 반영된다.

      2) 여러 선택자를 이용해서 같은 요소를 선택하더라도 지정된 css 속성이 다르면 모두 반영된다.

      1순위 : css 속성 : 속성값 !important;
      2순위 : inline-style 속성 (요소에 직접 작성되는 style 속성)
      3순위 : 아이디 선택자 (#아이디 속성명)
      4순위 : 클래스 선택자 (.class 속성명)
             -> html 태그 내부에 <style> </style> 태그를 이용해서 작성하는 방법
             -> html 태그 내부에 <link /> 태그를 이용해서 외부에 작성한 .css 파일을 불러오는 방법
      5순위 : 태그 선택자 (태그명)
